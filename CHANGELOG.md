## [0.0.1] - 2026-01-17

### Added
- Initial release
- Project initialization with `/new-project` command
- Roadmap creation with `/create-roadmap` command
- Phase planning with `/plan-phase` command
- Plan execution with `/execute-plan` command
- Progress tracking with `/progress` and `/status` commands
- Work session management with `/pause-work` and `/resume-work`
- Phase management: add, insert, remove phases
- Research and discovery modes
- User acceptance testing guidance with `/verify-work`
- Milestone management
- NEAR AI integration for model-agnostic LLM support
- Deferred issues tracking
- Assumption surfacing for phase approaches
